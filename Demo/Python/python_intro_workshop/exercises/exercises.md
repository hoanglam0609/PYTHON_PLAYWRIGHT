Exercises — Python Intro Workshop

1. Environment
- Verify Python version: `python3 --version`
- Create and run `hello.py` that prints a greeting.

2. Basics
- Create variables: name (string), age (int), balance (float). Print them.
- Ask user for their name and greet them.

3. Control Flow
- Write a program that asks for a number and prints whether it's even or odd.
- Write a `for` loop that prints numbers 1–5.

4. Functions
- Write a function `greet(name)` that returns a greeting string.
- Write a function that adds two numbers and returns the result.

5. Collections
- Create a list of 3 favorite foods and print them with indexes.
- Create a dictionary for a contact (name, phone) and print the phone.

6. Error Handling
- Modify the number input program to handle non-numeric input using `try/except`.

7. Practical
- Run `src/expense_tracker.py` and add three expenses.
- List expenses and show total.
- Remove an expense and verify the list updates.

Each exercise should be attempted live; encourage pairs to help each other.
