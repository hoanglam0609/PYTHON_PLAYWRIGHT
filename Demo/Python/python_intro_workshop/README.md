Python Introduction Workshop
===========================

Overview

A beginner-friendly 60-90 minute hands-on workshop introducing Python 3.12. The workshop covers basic Python concepts and walks participants through building a simple command-line "Personal Expense Tracker" application.

Prerequisites

- Python 3.12+ installed
- Terminal or command prompt

Quick start

1. Open a terminal in this folder.
2. Run the example app:

```bash
python3 src/expense_tracker.py
```

What you'll find

- `src/expense_tracker.py`: The demo CLI app for the practical section.
- `exercises/`: Practice exercises for each section.
- `solutions/`: Solutions to the exercises.
- `slides.md`: Presentation outline and speaker notes.

Workshop structure

1. Introduction to Python
2. Environment setup
3. Python basics (variables, types, I/O)
4. Control flow
5. Functions
6. Collections
7. Error handling
8. Practical demo: Expense Tracker

How to use this material

- Present each section with the slides and speaker notes in `slides.md`.
- Use `exercises/` after each section to give participants small tasks.
- Share `solutions/` for troubleshooting and reference.
